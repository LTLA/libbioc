Detailed Description

template<bool row_, typename Value_, typename Index_>
struct tatami::DirectApplyConfiguration< row_, Value_, Index_ >

Configuration for direct apply.

Set up the iteration across a Matrix in the specified iteration dimension. More specifically, we consider each matrix to be a collection of equi-length target vectors, where we wish to compute a statistic for each target vector. We do so directly by iterating over the target vectors, ignoring any preference in Matrix::prefer_rows(). This is useful when a statistic cannot be computed in a running manner.

Template Parameters
row_Whether each row is a target vector, e.g., to compute row-wise statistics.

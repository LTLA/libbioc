Detailed Description

template<bool RIGHT, int MARGIN, typename T = double, class V = std::vector<T>>
struct tatami::DelayedSubtractVectorHelper< RIGHT, MARGIN, T, V >

Subtract a vector from the rows/columns of a matrix, or vice versa.

This should be used as the OP in the DelayedIsometricOp class.

Template Parameters
TType to be returned after subtraction.
RIGHTShould the vector be subtracted from the matrix rows/columns? If false, the matrix rows/columns are instead subtracted from the vector.
MARGINDimension along which the subtraction is to occur. If 0, each element of the vector is assumed to correspond to a row, and the same value is subtracted from all entries in the same row of the matrix. If 1, each element of the vector is assumed to correspond to a column instead.
VClass of the vector holding the values to use in the subtraction.

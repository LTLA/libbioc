Detailed Description

template<typename Index_>
struct tatami::OracleStream< Index_ >

Stream predictions from the oracle.

This provides a friendlier interface to the Oracle classes that manages the prediction buffer. Consumers of Oracle predictions can receive and process one prediction at a time via the next() method, without worrying about the allocation and consumption of the buffer filled by Oracle::predict().

The OracleStream is primarily intended for Matrix developers to use within subclass implementations that can exploit oracular predictions. Users should not have to construct OracleStream instances.

◆ back()

template<typename Index_ >
bool tatami::OracleStream< Index_ >::back ( )

Move the state backwards by one prediction. This is guaranteed to work at least once after calling next(); multiple calls may or may not work, depending on whether the previous state has already been discarded. It is most useful for loops that need to inspect the next prediction before deciding whether to use it.

Returns
Whether or not the state was moved back successfully. This is guaranteed to be true once after a call to next().

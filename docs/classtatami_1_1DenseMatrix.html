Detailed Description

template<bool ROW, typename T, typename IDX = int, class V = std::vector<T>>
class tatami::DenseMatrix< ROW, T, IDX, V >

Dense matrix representation.

Template Parameters
ROWWhether this is a row-major representation. If false, a column-major representation is assumed instead.
TType of the matrix values.
IDXType of the row/column indices.
VVector class used to store the matrix values internally. This does not necessarily have to contain T, as long as the type is convertible to T. Methods should be available for size(), begin(), end() and []. If a method is available for data() that returns a const T*, it will also be used.

Detailed Description

template<typename T, typename IDX>
struct tatami::sparse_range< T, IDX >

A range of sparse values.

More specifically, this class defines a range of sparse (i.e., "non-zero") values, e.g., along a slice of a row or column. The aim is to hold (pointers to) the values and row/column indices, as well as the number of non-zero values within this range. This is most commonly returned by typed_matrix::sparse_row() and typed_matrix::get_sparse_column() methods.

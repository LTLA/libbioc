Detailed Description

template<typename T = double, bool RIGHT = true, int MARGIN = 0, class V = std::vector<T>>
struct tatami::DelayedDivideVectorHelper< T, RIGHT, MARGIN, V >

Divide the rows/columns of a matrix by a vector, or vice versa.

This should be used as the OP in the DelayedIsometricOp class.

Template Parameters
TType to be returned after division.
RIGHTShould the matrix rows/columns be divided by the vector? If false, the vector is instead divided by the matrix rows/columns.
MARGINDimension along which the division is to occur. If 0, each element of the vector is assumed to correspond to a row, and all entries in the same row of the matrix are divided by the same vector entry. If 1, each element of the vector is assumed to correspond to a column instead.
VClass of the vector holding the values to use in the division.

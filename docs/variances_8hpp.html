Detailed Description

Compute row and column variances from a tatami::Matrix.

Function Documentation

◆ column_variances()

template<typename Output = double, typename T , typename IDX >
std::vector< Output > tatami::column_variances ( const Matrix< T, IDX > *  p)

This uses the usual algorithm for matrices where tatami::Matrix::prefer_rows() is false, otherwise it uses Welford's algorithm. As a result, the computed variances will be slightly different (within numerical precision) for row- and column-major matrices.

TType of the matrix value, should be numeric.

Compute a running mean and variance from sparse data using Welford's method. This considers a set of "target vectors" that grow by one with each call to this function; the mean and variance of each target vector is computed.

Template Parameters
TType of the input data.
IDXType of the indices.
OType of the output data.
NzType fo the non-zero counts.
Parameters
rangeA SparseRange object specifying the number, indices and values of all non-zero elements. Each element is assigned to a target vector based on its index.
[out]meansPointer to an array containing the running means for each target vector.
[out]varsPointer to an array containing the running sum of squared differences from the mean for each target vector.
[out]nonzerosPointer to an array containing the running number of non-zero values for each target vector.
countNumber of times this function has already been called.
Returns
means and vars are updated with the corresponding elements from range. count is incremented by 1.

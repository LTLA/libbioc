Detailed Description

template<typename T>
class tatami::NakedArray< T >

Wrapper around a raw pointer to an array.

This serves as a drop-in replacement for some templated uses of std::vector in tatami classes. The idea is to enable use of tatami functions while avoiding unnecessary copies, provided that the raw pointer outlives the instance of the tatami class created from it.

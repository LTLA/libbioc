Detailed Description

template<int MARGIN, typename T = double, class V = std::vector<T>>
struct tatami::DelayedMultiplyVectorHelper< MARGIN, T, V >

Multiply a vector along the rows or columns of a matrix.

This should be used as the OP in the DelayedIsometricOp class.

Template Parameters
TType to be returned after multiplication.
MARGINDimension along which the multipication is to occur. If 0, each element of the vector is assumed to correspond to a row, and the same value is multiplied to all entries in the same row of the matrix. If 1, each element of the vector is assumed to correspond to a column instead.
VClass of the vector holding the values to use for multiplication

Detailed Description

template<typename Value, typename Index>
struct tatami::SparseRangeCopy< Value, Index >

A range of a sparse vector with copying.

This class defines a range along a sparse vector, where the values and indices of the structural non-zero elements are copied into internal vectors. It provides more memory safety than the SparseRange class, at the cost of extra allocation and copying.

Template Parameters
ValueData value type, should be numeric.
IndexRow/column index type, should be integer.

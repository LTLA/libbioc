Detailed Description

template<typename T, typename IDX>
struct tatami::SparseRangeCopy< T, IDX >

A range of a sparse vector with copying.

This class defines a range along a sparse vector, where the values and indices of the non-zero elements are copied into internal vectors. It provides more memory safety than the SparseRange class, at the cost of extra allocation and copying.

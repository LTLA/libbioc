Detailed Description

template<typename Id_, class ChunkContents_>
class tatami::LruChunkCache< Id_, ChunkContents_ >

LRU cache for chunks.

Template Parameters
Id_Type of chunk identifier, typically integer.
ChunkContents_Class containing the contents of a single chunk.

Implement a least-recently-used cache for chunks. This is typically used for Matrix representations where the data is costly to load (e.g., from file) and no oracle is provided to predict future accesses. In such cases, chunks of data can be loaded and cached such that any possible future access to an already-loaded chunk will just fetch it from cache.

Returns
Reference to a chunk. If the chunk already exists in the cache, it is returned directly. If the chunk does not exist and there is still space in the cache, a new chunk is created and populated with the contents of chunk id. If the chunk does not exist and there is no space in the cache, the least recently used chunk is evicted and its ChunkContents_ is populated with the contents of chunk id.

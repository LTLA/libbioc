bool tatami::Options::sparse_extract_index = true

Whether to extract the indices in SparseExtractor::fetch. If set to false, this can be used to avoid unnecessary computation and copying in Matrix methods.

◆ sparse_extract_value

bool tatami::Options::sparse_extract_value = true

Whether to extract the values in SparseExtractor::fetch. If set to false, this can be used to avoid unnecessary computation and copying in Matrix methods.

◆ sparse_ordered_index

bool tatami::Options::sparse_ordered_index = true

Whether the structural non-zeros returned by SparseExtractor::fetch should be ordered by increasing index. Setting this to false may reduce computational work in situations where the order of non-zero elements does not matter.

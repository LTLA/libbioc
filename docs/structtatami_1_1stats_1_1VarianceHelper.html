Compute the mean and variance along a vector. This avoids a redundant calculation of the mean as it is already computed as part of the variance calculation.

Template Parameters
TType of the input data.
Parameters
ptrPointer to an array of values of length n.
nSize of the array, must be greater than 1.
bufferUnused, provided here for consistency only.
Returns
The sample mean and variance of values in [ptr, ptr + n). If n == 0, the mean is set to NaN, and if n < 2, the variance is set to NaN.

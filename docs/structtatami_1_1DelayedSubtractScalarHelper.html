Detailed Description

template<typename T = double, bool RIGHT = true>
struct tatami::DelayedSubtractScalarHelper< T, RIGHT >

Subtract a scalar from all values of a matrix, or vice versa.

This should be used as the OP in the DelayedIsometricOp class.

Template Parameters
TType to be returned after subtraction.
RIGHTShould the scalar be subtracted from the matrix value? If false, the matrix value is subtracted from the scalar.

Detailed Description

template<DelayedArithOp op_, bool right_, int margin_, typename Value_ = double, typename Vector_ = std::vector<double>>
struct tatami::DelayedArithVectorHelper< op_, right_, margin_, Value_, Vector_ >

Delayed vector arithmetic.

This should be used as the OP in the DelayedUnaryIsometricOp class.

Template Parameters
op_The arithmetic operation.
right_Whether the vector's values should be on the right hand side of the arithmetic operation. Ignored for some op_.
margin_Matrix dimension along which the operation is to occur. If 0, each element of the vector is assumed to correspond to a row, and that value is subtracted from all entries in the same row of the matrix. If 1, each element of the vector is assumed to correspond to a column instead.
Value_Type of the data value.
Vector_Type of the vector.

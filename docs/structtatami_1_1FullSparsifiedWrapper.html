Detailed Description

template<bool oracle_, typename Value_, typename Index_>
struct tatami::FullSparsifiedWrapper< oracle_, Value_, Index_ >

Wrap a full dense extractor in the sparse interface.

This can be used to quickly implement the full sparse extraction methods for a dense Matrix subclass. The dense extraction is performed as usual and every value is treated as a structural non-zero; this wrapper just adds the associated indices to satisfy the sparse extraction interface.

oracle_Whether an oracle is involved.

Index_Type of the row/column indices.
